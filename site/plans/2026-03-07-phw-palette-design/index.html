    Design: PHW-Aligned Accessible Mermaid Palette¶

    Date: 2026-03-07 Status: Approved Branch: fix/mermaid-contrast-and-linebreaks

    Problem¶

    All Mermaid diagrams across the 18 docs pages use light pastel fills (e.g. #fee2e2, #fef3c7, #eff6ff) with near-black text (color:#1f2937). In MkDocs Material dark mode these become blinding bright spots on a dark page background. Additionally:

    • Light pastels fail several colorblindness simulations (deuteranopia, protanopia)
    • No CSS-only fix is viable — Mermaid renders inline SVG with inline style="" attributes that cannot be overridden by external CSS without !important on every property
    • The site uses MkDocs Material's deep purple theme, which does not align with Public Health Wales corporate branding

    Decisions¶

    Approach chosen: Global semantic palette replacement¶

    Replace all hardcoded fill: and color: values in every Mermaid style directive with a six-role semantic palette derived from the PHW corporate brand. Add docs/stylesheets/extra.css to apply PHW brand colours to the MkDocs theme and to handle any un-styled Mermaid nodes in dark mode.

    CSS-only and per-diagram %%{init}%% approaches were rejected because: - CSS cannot override inline SVG styles reliably - %%{init}%% blocks don't address existing style directives and add verbose boilerplate

    Palette Specification¶

    All six roles use white (#ffffff) text. All pass WCAG AA for large text (≥ 18px or ≥ 14px bold), which covers typical Mermaid node labels.

    Role Hex WCAG (white text) Semantic meaning
    PHW Magenta #c60158 5.76:1 ✓ Start node / key concept / brand highlight
    PHW Navy #325083 7.08:1 ✓ Process step / info / neutral
    Danger #9b1c1c 6.0:1 ✓ Bad / old / before / error
    Success #065f46 7.8:1 ✓ Good / new / after / done
    Warning #92400e 6.5:1 ✓ Caution / highlight / in-progress
    Teal #0d9488 4.0:1 (large ✓) Cloud / deployed / output

    PHW brand source: Extracted from live site CSS at phw.nhs.wales (March 2026). Primary magenta #c60158 is used for primary buttons and CTAs site-wide. Navy #325083 is used for navigation and secondary elements.

    Colorblind strategy¶

    Colour is supplementary — every Mermaid node carries a text label that independently conveys its semantic meaning. No meaning is ever communicated by colour alone. Danger/Success are also distinguishable by luminance difference and by shape context within each diagram.

    Font¶

    Inter (current) is retained. It is the closest freely licensed web equivalent to Frutiger, the NHS/PHW print typeface, and requires no licensing.

    Acceptance Criteria¶

    • mkdocs build --strict passes with zero warnings
    • Nav bar renders in PHW Magenta (#c60158) in both light and dark mode
    • All Mermaid nodes use one of the six palette colours
    • No color:#1f2937 remains in any live docs page
    • No pastel fills (#fee2e2, #fef3c7, #eff6ff, #f0fdf4, #e0e7ff, #ede9fe, #fce7f3, #d1fae5) remain in any live docs page
    • Visual inspection in dark mode confirms no bright-box contrast issues
